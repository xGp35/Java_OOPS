# Java OOPS Projects

This workspace contains Java practice projects focused on object-oriented programming concepts.

## Projects

- `Dog_Array/` - Examples using arrays and classes with `Dog`, `Dog2`, and `GoodDog`.
- `Guessing_Game/` - A number guessing game with `GuessGame` and `Player` classes.
- `Head_First/` - Exercises from the Head First Java book, including `DrumKit`, `PhraseOMatic`, and `Shuffle_game`.
- `Random/` - OOP and Java concept examples such as composition, inheritance, aggregation, and more.
- `SimpleStartup/` - A simple startup game project with helper classes.
- `StartupBust/` - Startup-themed Java game implementation.

## How to run

From the `Java_OOPS` directory, compile and run Java files using standard commands.

```bash
javac <path-to-file>.java
java <MainClass>
```

For example, to run the guessing game:

```bash
cd Guessing_Game
javac GuessGame.java Player.java
java GuessGame
```
